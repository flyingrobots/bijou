# Graceful Lowering Preserves Meaning

Rich mode is not the only reality.

Lowering does not need visual parity, but it must preserve truthful meaning across:

- rich / interactive
- static
- pipe
- accessible

Implications:

- lower modes should remain explicit and usable
- color, motion, and layout should not be the only carriers of meaning
- inaccessible “TUI-only” experiences should be treated as debt, not victory
