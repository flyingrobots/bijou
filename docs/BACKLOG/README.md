# Backlog

This directory is the source of truth for proposed work that has **not** started yet.

## Rules

- one file per backlog item
- filename format: `<Legend>-<id>-<slug>.md`
- backlog items may be rough and incomplete
- backlog items can capture:
  - feature work
  - design debt
  - tech debt
  - COOL IDEAS

When a cycle starts, move the file itself into `docs/design/` and enrich it there.
