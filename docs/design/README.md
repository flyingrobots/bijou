# Design Cycles

This directory holds active and landed cycle docs.

## Lifecycle

- work starts in `docs/BACKLOG/`
- when a cycle begins, move the file here
- enrich it with the information needed to implement the cycle
- keep the design doc as the historical record of the cycle once it lands

## Required sections

- linked legend
- human users, jobs, and hills
- agent users, jobs, and hills
- human playback
- agent playback
- linked invariants
- implementation outline
- tests to write first
- retrospective
