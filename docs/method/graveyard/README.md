# Graveyard

Superseded or abandoned METHOD ideas live here as historical evidence.

Use `graveyard/` only when the original idea is no longer the right live
work item: it was killed, replaced, or narrowed into a different follow-on.

Do not put finished shipped work here. Finished work belongs in
`docs/method/retro/`.

Do not use this directory to determine current work. Start with open GitHub
Issues and their `lane:*` labels.
